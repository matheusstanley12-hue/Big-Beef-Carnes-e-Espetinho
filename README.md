# Big Beef Carnes e Espetinho - Sistema de Gestão

Sistema completo de gestão para o estabelecimento, com foco em autoatendimento, controle de produção, gestão de equipe e frente de caixa.

## 🚀 Funcionalidades Principais

- **📍 Autoatendimento**: Clientes podem abrir o cardápio e realizar pedidos.
- **🍳 Painel de Produção**: Cozinha e Bar com controle de status.
- **📋 Gestão de Equipe**: Controle de acesso por níveis.
- **💰 Frente de Caixa**: Fechamento de contas.
- **📊 Dashboard do Dono**: Visão em tempo real de faturamento.

## 🛠️ Tecnologias Utilizadas

- **Frontend**: React + TypeScript + Vite
- **Estilização**: CSS Vanilla (Design Premium/Dark Mode)
- **Backend/DB**: Supabase (Auth, Database)
- **Estado**: Zustand 
- **Roteamento**: React Router DOM

## 📦 Como rodar o projeto

1. Instale as dependências:
   ```bash
   npm install
   ```
2. Configure as variáveis de ambiente no `.env`:
   ```env
   VITE_SUPABASE_URL=seu_url
   VITE_SUPABASE_ANON_KEY=sua_chave
   ```
3. Inicie o servidor:
   ```bash
   npm run dev
   ```
